# FirstProject

Angular project made on <a href="https://www.udemy.com/the-complete-guide-to-angular-2/">Angular courses</a> .

<br>

Features: <br>
-Responsive design <br>
-Login/Register <br>
-Save new recipes <br>
-Delete new recipes <br>
-Edite recipes <br>
-Save/Edit/Delete ingredients of recipes<br>
